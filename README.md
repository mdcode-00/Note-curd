# Note-curd
A full-stack Notes Application built using the MERN stack (MongoDB, Express, React, Node.js) to practice and learn CRUD (Create, Read, Update, Delete) operations.
This project helps understand how frontend and backend interact, how to manage APIs, and how to store and retrieve data from MongoDB.
